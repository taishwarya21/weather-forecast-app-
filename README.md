# Weather Forecast App

A weather application that provides real-time weather information using a public weather API.

## Features
- Search weather by city
- Real-time temperature and weather details
- Dynamic UI updates
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- REST API

## API Used
OpenWeatherMap API

## Learning Outcomes
- API integration
- Fetch API and asynchronous JavaScript
- JSON data handling
- Error handling

## Future Improvements
- 5-day forecast
- Current location weather
- Dark mode support
